- Open Command Palette: ```Ctrl + Shift + P```
- Comment Code: ```Ctrl + K, Ctrl + C```
- Uncomment Code: ```Ctrl + K, Ctrl + U```
- Toggle Comment/Uncomment Code: ```Ctrl + /```
- Format Selection: ```Ctrl + K, Ctrl + F```
- Format Document: ```Shift + Alt + F```
