### Installation
- [Install the Dapr CLI](https://docs.dapr.io/getting-started/install-dapr-cli/)
  ```
  dapr
  dapr -v
  dapr --version
  ```
- [Initialize Dapr](https://docs.dapr.io/getting-started/install-dapr-selfhost/)
  ```
  dapr init
  dapr --version
  docker ps
  ```
